Variables

A variable is a location in memory which has data allocated to it so that it can be used and manipulated by the user.

Constants

A constant is like a variable although the value of the constant is not able to change while the program is running unlike a variable.

Data types

The data stored within a variable or a constant can be of different types, when the program is running the type of data a variable or constant uses cannot change.

Integer

An integer stores a number with no decimal point e.g. 1

Boolean

A boolean stores either a true or false value e.g. Is the door open? True or false?

Real

A real value is a value with a decimal point this can be numbers like 1.45654 or 9012.2 etc.

Character

A character is a single symbol which represents a letter or number etc. such as 'a'.

String

A string stores combinations of characters, this can be a word, numbers or a mixture of the two for example a password.
